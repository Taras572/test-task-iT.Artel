    *ngIf видаляє/додає елемент з DOM дерева, [hidden] візуально приховує елемент але він присутній в DOM дереві

Node JS

  1. Що таке файл package.json?

    1 package.json це файл в якому зберігається список npm модулів які нам потрібно встановити за допомогою команди npm install, це потрібно для того щоб не завантажувати на віддалений репозиторій всі node_modules, за допомогою списку ми встановимо потрібні версії модулів на іншому робочому місці

  2. Які є типи запитів HTTP?

    Oсновні запити:
    GET запитує дані з сервера
    POST надсилає дані на сервер
    PUT надісилає лише оновлені дані на сервер
    DELETE видаляє дані з сервера

  3. Поясніть різницю між локальним і глобальним встановленням пакетів npm

    Локальне встановлення npm модулів це коли модуль встановлюється в директорію node_modules та додається до файлу package.json кожного разу після завантаження проекту з віддаленого репозиторію нам потрібно встановити модулі із списку файлу package.json на робоче місце

    Глобальне встановлення npm модулів це коли модуль встановлюється в глобальний каталог, модуль не додається до файлу package.json модуль завжди буде йти у парі з проектом, тому його не потрібно буде встановлювати заново.

  4. Поясніть порядок виконання прослуховувачів подій у Node.js. Що буде результатом цього коду і чому?

    Результат коду:
    Statement A
    Emitted Statement - FIRST
    Emitted Statement - SECOND
    Statement B

    eventEmitter.on('myEvent', (data) => {
    console.log(data, '- FIRST');
    });
    перший прослуховувач виконається швидше оскільки він додається раніше ніж другий
